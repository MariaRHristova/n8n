# n8n Workflows

This repository contains my exported n8n workflows in an organized structure.

## What is included

- `workflows/` with the JSON exports grouped by topic
- `workflows/manifest.csv` for mapping original export names to their new paths
- `docs/screenshots/cloud-folders/` with the reference screenshots I used while organizing the repo

## Folder Layout

- `00-intro-and-foundations`
- `01-integrations-and-webhooks`
- `02-data-and-transformations`
- `03-ai-and-agents`
- `04-reliability-and-monitoring`
- `05-mcp`
- `06-content-pilot`
- `07-t4-exercises`
- `08-projects`

The goal is to keep the workflows easy to browse while still presenting the work in a clean GitHub repository.
